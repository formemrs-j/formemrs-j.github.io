基本数据类型

数字整型，浮点

  • python number数据类型包含整型(int)和浮点型(float),float不区分单双精度，默认为双精度

  • 查看输入的数据类型，用函数type()

    >>> type(1)
<class 'int'>
>>> type(1*1.0)
<class 'float'>
>>> type(1/1)
<class 'float'>
>>> type(1//2)
<class 'int'>
>>>

  • python 乘法用* 除法用/(除法返回为float) 强制转换为返回整型除法(int)用//

各进制之间的表示与转换

  • python 表示二进制在前面加上特殊标识符0b 例如0b10

  • 表示八进制的特殊符号为0o例如0o10

  • 表示十六进制的特殊符号为0x例如0x10

  • 强制进制转换二进制为bin(),八进制为oct(),十进制int(),十六进制为()

字典

  • 以{key:value,key1:value1}这种形式的数据类型为字典，一个key对应一个value,key不能重复
  • 访问字典中的元素必须用key来访问，key必须为不可变的数据类型，例如，int,str,tuple
  • 空的字典用{}来定义

算术运算符

  • 常见的+，-，*，/都可以
  • 乘方用**，除法取整用//，取余数用%

赋值运算符

  • 先赋值后运算
  • 例如a+=1,就相当于a=a+1,其他类似

比较运算符（关系运算符）

  • 返回的是布尔类型
  • 不仅仅是数字才能比较，字母（转换成ascii码比较）,元组，列表按序号一个个比较

成员运算符

  • in 例如 1in [1,2,3] 返回True
  • not in 是否不在
  • 返回的都是布尔类型

身份运算符

  • is 例如 1 is 1.0 返回False（比较的是内存地址）
  • 有别于== ，比较的是值

位运算符

  • &按位与 计算规则 先把数字转换为二进制一位位比较，如果都是1，返回1，否则返回0，再将计算结果转换为十进制输出
  • |按位或 计算规则 先把数字转换为二进制一位位比较，如果有一位是1，返回1，否则返回0，再将计算结果转换为十进制输出

错误分析

a = input()
if a == 1:
    print("apple")
elif a== 2:
    print("banana")
else:
    print("peach")
  • 错误原因，代码无论输入什么都是输出“peach”，因为输入的是字符串，而判断的是数字

改进

a = input()
a=int(a)
if a == 1:
    print("apple")
elif a== 2:
    print("banana")
else:
    print("peach")
  • 添加语句 a=int(a)，强制转换成整数型
  • 在进行简单判断时，不要思维定势，可以用简单的逻辑运算符代替

循环语句

while

counter = 1

while counter <= 10:
    counter+=1
    print(counter)
else:
    print("结束")
  • while循环容易出现死循环，一般用for代替

for in

a=[["apple","banana","peach"],(1,2,3)]
for x in a:
    for y in x:
        print(y)
else:
    print("end")
  • for in常用于遍历字典 列表
  • for in 可以配合else使用

for in range()

  • 指定次数循环
for x in range(1,10):
    if x == 3:
        break
    print(x)
  • 可以搭配break使用，直接跳出循环
  • 也可以配合continue使用,除了判断的这个条件不执行，其他继续执行
for x in range(1,10):
    if x == 3:
        continue
    print(x)

包，模块，函数，变量的作用域

python项目的组织结构

  • 包，模块，类，函数，变量

导入模块

  • 方法一
  • import方法
import t.c4 
print(t.c4.a)
print(t.c4.b)
  • 方法二
  • 简化写法 import as
  • 将模块名在引用时重命名达到简写的目的
import t.c4 as w
print(w.a)
print(w.b)
  • 方法三
  • from 模块 import * 导入包内所有变量，也可引入单个变量
from t.c4 import *
print(a)
print(b)
  • 方法四
  • 方法三的加强，在要导入的模块添加“__all__”变量,限制引入的变量
__all__=["a","b","c"]
a = 1
b = 2
c = 3
  • 方法五
  • from 包 引入 模块
from t import c4
print(c4.a)
print(c4.b)
print(c4.c)
  • 包文件中有一个__init__.py的头文件，导入包时会优先执行头文件中的内容
  • 可以在头文件中添加all变量来限制导入的内容
  • 导入包时禁止循环导入，python会自动报错
__all__=["a"]
a = 1
  • 方法六
  • 引入多个变量可以使用都好隔开
from t.c4 import a,b,c 
print(a)
print(b)
print(c)

形参和实参

  • 函数中定义的参数为形式参数
  • 调用时传入的参数为实际参数

必须参数和关键字参数

  • 必须参数函数执行时必须要传递的函数
  • 关键字参数可以解决调用时参数的位置
def add(x,y):#x,y即为形式参数
    result1=x+y
    result2=x*y
    return result1,result2
a,b = add(y=1,x=2)#1,2为实际参数
print(a,b)

默认参数

  • 即在函数调用中需要重复传入的参数可以使用默认参数
def student_file(name,gender="男",age="18",college="人民路小学"):
    print("我叫"+name)
    print("性别"+gender)
    print("我今年"+age+"岁")
    print("我在"+college+"上学")
student_file("卢本伟")
  • 默认参数可以修改，在调用时传入即可
